1.磁共振图像介绍

磁共振成像（Magnetic Resonance Imaging，简称MRI）是一种利用强磁场、无线电波和计算机技术结合的医学成像技术。它能够提供人体内部结构的详细图像，尤其是软组织，如大脑、脊髓、关节、肌肉和其他内脏器官。以下是关于MRI的一些关键点介绍：

  1. 成像原理：MRI基于原子核在磁场中的磁性质。当身体被置于一个强磁场中时，某些原子核（如氢原子核）会与磁场对齐。无线电波脉冲被用来扰动这些原子核，当无线电波停止后，原子核会返回其原始状态，并在这个过程中释放出能量。这些释放出的能量被MRI机器检测到，并用来生成图像。

  2. 软组织对比：MRI特别擅长区分不同的软组织类型，如肌肉、肌腱、韧带、大脑组织等，这使得它在神经系统疾病的诊断中非常有用。

  3. 无辐射：与X射线或CT扫描不同，MRI不使用辐射，因此被认为是一种安全的成像方法。

  4. 多平面成像：MRI可以生成多个平面的图像，包括轴向、冠状和矢状面，这有助于从不同角度查看内部结构。

  5. 功能性MRI（fMRI）：这是一种特殊的MRI技术，它可以检测大脑活动的变化，通常用于研究大脑功能和神经精神疾病。

  6. 对比剂使用：在某些情况下，可能需要使用对比剂来提高图像的清晰度，尤其是在检测血管、肿瘤或其他异常时。

  7. 局限性：MRI扫描时间较长，成本较高，且对于有心脏起搏器、金属植入物或某些其他医疗条件的患者不适用。

  8. 临床应用：MRI广泛应用于各种临床情况，包括但不限于脑部疾病（如中风、肿瘤、癫痫和阿尔茨海默病）、脊髓损伤、关节和骨骼问题、心脏病和肿瘤分期。

2.磁共振图像增强模型与算法研究

磁共振图像增强是医学图像处理领域的一个重要研究方向，其目的是改善MRI图像的质量，使之更适合于临床诊断和医学研究。图像增强可以提高图像中某些特征的可见性，比如对比度、边缘、纹理等，同时尽量保留图像的原始信息。以下是一些常见的磁共振图像增强模型与算法：

  1. 空间域滤波器：

    • 平滑滤波器：如高斯滤波器，用于减少图像噪声，但可能模糊图像细节。
    • 锐化滤波器：如拉普拉斯滤波器，用于增强图像边缘，提高图像的清晰度。

  2. 频率域滤波器：

    • 使用傅里叶变换将图像转换到频率域，然后对频率成分进行调整，以增强或抑制特定的频率带。

  3. 对比度增强：

    • 直方图均衡化：调整图像的直方图，使得像素值分布更加均匀，从而增强图像的全局对比度。
    • 对比度限制自适应直方图均衡化（CLAHE）：是一种改进的直方图均衡化方法，它在小区域内增强对比度，减少噪声放大。

  4. 非线性映射：

    • 对数变换和幂律变换：这些变换可以增强图像的暗区域或亮区域，改善图像的局部对比度。

  5. 深度学习技术：

    • 卷积神经网络（CNN）：可以设计用于特定类型的图像增强任务，如去噪、超分辨率或特征增强。
    • 生成对抗网络（GAN）：通过对抗过程生成更加逼真和高质量的图像。

  6. 多模态融合：

    • 结合不同成像参数或不同时间点获取的MRI图像，以提高图像的信息量和诊断价值。

  7. 特定结构增强：

    • 对于特定的医学应用，如血管成像、肿瘤检测等，可以设计算法特别增强这些结构的特征。

  8. 图像分割和特征提取：

    • 利用图像增强技术辅助图像分割，准确区分不同的组织类型，提取有用的医学特征。

  9. 优化算法：

    • 利用优化技术，如梯度下降，来调整图像增强过程中的参数，以达到更好的视觉效果或满足特定的诊断需求。

  10. 质量评估：

    • 在增强过程中，需要对图像质量进行评估，以确保增强后的图像在视觉上和临床上都是可接受的。

在研究和开发MRI图像增强模型与算法时，需要考虑算法的计算效率、增强效果、以及对诊断准确性的影响。此外，还需要进行大量的实验验证，以评估增强算法在不同类型MRI图像和不同临床情况下的表现。随着技术的进步，深度学习等先进的机器学习技术在MRI图像增强领域展现出巨大的潜力。

3.介绍下论文

背景

AD没法根治，因此通过分类算法辅助医生提高其早期诊断率是非常重要的。医生主要是通过MRI来判断分类，比如关键的组织像海马体、内嗅皮层。

目的

提高AD分类的准确度的同时，保证AD分类的效率

数据集

使用的是ADNI（阿尔茨海默病神经影像学研究项目）、使用了220个CN（认知正常）、310个MCI（早期认知障碍）、255个AD（阿尔茨海默病）

相关的数据集

  1. OASIS (Open Access Series of Imaging Studies)

  2. The Australian Imaging, Biomarker & Lifestyle (AIBL) Study

  3. The National Alzheimer’s Coordinating Center (NACC) Database 由美国阿尔茨海默病协会维护

方法

为了同时集成CNN和vit各自优点

  1. CNN优点是需要计算资源少提取特征比较快，但在捕捉图像的全局依赖关系方面不足。

  2. vit优点是通过全局自注意力机制，ViT能够捕捉图像中任意两个像素之间的依赖关系，无论它们之间的距离有多远。并且可以高效地并行化。但占用计算资源、模型训练速度慢

LP： 先使用CNN和maxpool提取一次特征并减少特征图大小之后再进行分块操作可以减少模型参数量。并使用了较少的patch（模型捕捉到图像的细粒度特征，这对于检测图像中的细节变化非常重要。有助于学习到更精细的图像表示。）之后通关vit进行分类。

DWLA： 动态局部窗口注意力机制。允许注意力只在窗口内部进行计算，将注意力计算图划分成一个个窗口，窗口大小会动态改变。这种方法允许模型在不同阶段关注不同尺度的特征（7个transformer Encoder块中动态改变,从大窗口到小窗口）

相比于大窗口计算，小窗口计算的好处

Loss ： 加权损失函数，权重系数是0.1。
center loss: 一方面。有些人正常衰老也会出现一定程度的脑萎缩。导致CN和MCI的MRI图像比较接近。另一方面，有些AD病人可能萎缩的部位不一样，从而导致同为AD的MRI图像相差很大。
中心损失思想： 通过强迫每个类别的特征尽可能接近类别的中心，来提高对特征的区分能力。这有助于区分不同类别中的特征，减少了同一类别内的特征离散度，从而提高了分类模型的准确性和鲁棒性。

中心损失的计算：中心损失函数通常采用平方距离作为距离度量，并希望这个距离尽可能小。因此，中心损失可以表示为所有样本点到其对应类别中心点距离的总和。中心损失的一个关键点是类别中心的更新，这通常在模型训练过程中进行。

中心损失函数的实现涉及到额外的存储和计算开销，因为需要存储和更新每个类别的中心点。然而，它在人脸识别和细粒度图像分类中，已被证明是非常有效的。

Cross Entropy Loss : 它衡量的是模型预测的概率分布与真实标签的概率分布之间的差异。连续可微的，便于使用梯度下降等优化算法进行训练

对于二分类问题，交叉熵损失定义如下：

[ L = -\left( y \log(p) + (1 - y) \log(1 - p) \right) ]

其中：

  • ( y ) 是真实标签，其值为 0 或 1。
  • ( p ) 是模型预测为正类的概率。
  • ( L ) 是损失值。

标签平滑：为了避免模型对某些类别过于自信，有时会使用标签平滑技术，即真实标签不是完全确定的1或0，而是稍微平滑的值（例如0.9和0.1）。

Vision transformer

AlexNet

网络结构
AlexNet是一种深度卷积神经网络（CNN）架构，由Alex Krizhevsky、Ilya Sutskever和Geoffrey Hinton在2012年提出，并在当年的ImageNet大规模视觉识别挑战赛（ILSVRC）中取得了突破性的成绩。AlexNet的网络结构相对复杂，参数众多，是当时首个真正意义上的深度卷积神经网络，其深度是之前神经网络的数倍。

AlexNet的网络结构主要包括以下几个部分：

  1. 输入层：AlexNet的输入图像尺寸是224x224x3（RGB三通道）。原论文中提到，虽然实际图像尺寸为227x227x3，但通常使用224x224x3的尺寸进行处理。

  2. 卷积层：网络包含5个卷积层（C1至C5），每个卷积层后面紧跟一个ReLU激活函数和一个局部响应归一化（LRN）层。卷积层的配置如下：

    • C1：11x11x3卷积核，96个卷积核，步长为4。
    • C2：5x5x48卷积核，128个卷积核，步长为1，应用在C1层的输出上。
    • C3：3x3x256卷积核，384个卷积核，步长为1。
    • C4：3x3x192卷积核，192个卷积核，步长为1。
    • C5：3x3x192卷积核，128个卷积核，步长为1。

  3. 池化层：在第1、2、5个卷积层后面各有一个最大池化层，池化核大小为3x3，步长为2。

  4. 全连接层：在卷积和池化层之后，网络包含3个全连接层（FC6、FC7和FC8）。

    • FC6：4096个神经元。
    • FC7：4096个神经元。
    • FC8（输出层）：1000个神经元，对应ImageNet的1000个类别。

  5. Dropout层：在FC6和FC7层后应用了Dropout技术，以减少过拟合现象。

  6. softmax输出：最终的FC8全连接层输出通过softmax函数转换为概率值，用于图像分类。

AlexNet的创新之处包括使用ReLU激活函数、局部响应归一化（LRN）和Dropout技术，以及能够在GPU上进行并行训练的架构设计。此外，AlexNet在训练时采用了数据增强策略，通过对图像进行旋转、翻转、裁剪等变换来增加数据多样性。

局部响应归一化（Local Response Normalization，简称LRN）

LRN 操作涉及对每个特征图（feature map）中的元素进行归一化，使得较大响应的元素相对更大，而较小响应的元素相对更小。这样，LRN能够在不同通道上产生专注不同特征的feature map，从而减少不同卷积核产生的feature map之间的相关性

ReLU激活函数

在深度学习中，”Rule”激活函数并不是一个标准术语，因此可能存在一些误解或混淆。如果您是指ReLU（Rectified Linear Unit）激活函数，它是一种非常流行的激活函数，其公式如下：

[ \text{ReLU}(x) = \max(0, x) ]

ReLU激活函数的特点是当输入x为正数时，输出就是x本身；当输入x为负数时，输出为0。这种操作被称为”rectification”，它允许网络在正区间内线性地响应输入变化，而在负区间内则抑制输出。

ReLU激活函数优点

  1. 计算简单：ReLU激活函数的计算非常直接和简单，它仅在输入值为正时激活，这减少了计算量并加快了训练速度。

  2. 收敛速度快：相比于其他如sigmoid或tanh等激活函数，ReLU激活函数在正区间内是线性的，这有助于在训练过程中实现更快的梯度下降，从而加快收敛速度。

  3. 缓解梯度消失问题：在深度神经网络中，使用sigmoid或tanh激活函数时，梯度可能会随着层数的增加而迅速减小，导致梯度消失问题。ReLU由于其线性特性，在正区间内梯度恒定，这有助于缓解梯度消失问题。

  4. 稀疏激活：ReLU激活函数在负区间内输出为零，这意味着网络中的神经元会稀疏地激活，这有助于减少计算量并可能提高模型的泛化能力。

  5. 减少过拟合风险：稀疏激活的特性还意味着模型倾向于学习更加分散的表示，这可以减少过拟合的风险。

  6. 实现简单：ReLU激活函数的实现非常简单，易于在各种深度学习框架中实现和使用。

尽管ReLU激活函数有很多优点，但它也有缺点，如在训练初期可能会导致神经元的死亡（即在训练过程中某些神经元可能永远不会激活）。为了解决这个问题，变体如Leaky ReLU和Parametric ReLU被提出，以允许负输入有一个非零的梯度。

  • Leaky ReLU：允许负输入有一个非零的梯度，公式为 ( \text{LeakyReLU}(x) = \max(\alpha x, x) )，其中 ( \alpha ) 是一个很小的正数。
  • Parametric ReLU（PReLU）：Leaky ReLU的一个泛化，其中负输入的梯度 ( \alpha ) 是一个可学习的参数。
  • Exponential Linear Unit（ELU）：提供一个负输入非零输出的平滑曲线，公式为 ( \text{ELU}(x) = x ) if ( x > 0 ) else ( \alpha (e^x - 1) )。

总的来说，ReLU激活函数由于其简单、快速和有效的特点，在深度学习中得到了广泛的应用，尤其是在像AlexNet这样的深度卷积神经网络中。

CBAM

CBAM（Convolutional Block Attention Module）是一种用于深度卷积神经网络的注意力机制模块，它旨在提高网络对输入数据的特征表示能力。CBAM 是由 Sanghyun Woo 等人在 2018 年提出的，它可以作为一个即插即用的模块，嵌入到现有的卷积神经网络架构中，以增强特征的表达能力。

CBAM 包含两个主要部分：通道注意力（Channel Attention）和空间注意力（Spatial Attention）。这两个部分按顺序应用于卷积层的输出上，以逐步提炼特征表示。

  1. 通道注意力（Channel Attention）：

    • 通道注意力的目的是允许网络专注于更重要的通道特征，同时抑制不重要的通道。
    • 它首先使用全局平均池ing（Global Average Pooling, GAP）将每个通道的特征压缩成一个单独的数值，然后通过两个连续的全连接（FC）层，其中第二个FC层使用ReLU激活函数，最后通过一个sigmoid激活函数输出每个通道的重要性权重。

  2. 空间注意力（Spatial Attention）：

    • 空间注意力的目的是强调特征图中更重要的空间位置，同时忽略不重要的位置。
    • 空间注意力通过将通道注意力的输出输入到一个卷积层，然后应用ReLU激活函数，接着再使用一个3x3的卷积核进行卷积操作，最后通过sigmoid激活函数输出每个空间位置的重要性权重。

CBAM 的工作流程如下：

  • 输入一个卷积层的特征图。
  • 应用通道注意力来获得每个通道的权重。
  • 使用这些通道权重来线性组合原始特征图中的通道。
  • 将通道注意力的输出传递给空间注意力模块。
  • 应用空间注意力来获得每个空间位置的权重。
  • 使用这些空间位置权重来线性组合特征图中的空间位置。

CBAM 可以有效地增强网络对输入数据的感知能力，从而提高下游任务的性能，如图像分类、目标检测等。由于 CBAM 是一个轻量级模块，它不会显著增加计算负担，因此可以很容易地集成到各种深度学习架构中。

CBAM 的一个关键优势是它提供了一种简单而有效的方法来增强特征表示，而无需改变网络的基本架构。这使得它成为提高现有模型性能的有用工具。此外，CBAM 的概念也启发了后续许多其他注意力机制模块的发展。
