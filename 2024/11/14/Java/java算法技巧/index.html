当使用增强 for 循环时，循环变量（这里是 num）实际上是集合中元素的一个副本，而不是对集合中元素本身的直接引用。也就是说，在循环体内对 num 进行的任何修改操作（如 num++），都只会影响这个副本，而不会对集合中的原始元素产生任何影响。
