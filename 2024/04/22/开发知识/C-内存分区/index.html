C++内存分区

C++内存分区模型

程序运行前划分为代码区和全局区，程序运行后分为栈区和堆区

  • 全局区：全局变量，静态变量（static关键字），常量（字符串常量 和 const修饰的全局常量）
  • 栈区：由编译器自动分配释放，存放着函数的参数值，局部变量等，注意不要返回局部变量的地址
  • 堆区：由程序员分配释放，若程序员不释放，程序结束后将由操作系统回收

new运算符

  • 利用new运算符可以将数据开辟到堆区：int *p = new int(10)
  • new运算符返回的是数据在栈中的地址
  • 利用 delete 可以释放堆区数据（delete p ）

引用的基本使用

  • 作用：给变量起别名
  • 基本语法：数据类型 &别名 = 原名（ int &b = a ）
  • 引用一定要初始化，且之后不能再更改了
  • 函数参数的引用传递，形参也会修饰实参

类和对象

类的访问权限

类的默认访问权限是private

权限名 描述
public 类的内外都可以访问
protected 只有类内可以访问，且可以被继承
private 只有类内可以访问，且不可以被继承

成员属性私有化
1.可以自己控制读写权限
2.对于写可以检测数据的有效性

构造函数和析构函数

构造函数语法：名（）{ }

  • 1.构造函数，没有返回值也不写void
  • 2.构造函数名称应与类名相同
  • 3.构造函数可以有参数，也可以重载
  • 4.程序在调用对象时会自动调用构造函数

析构函数语法：（）{ }

  • 1.析构函数，没有返回值也不写void
  • 2.析构函数名称应与类名相同，在名称前加上 ~
  • 3.析构函数不可以有参数
  • 4.程序在对象被销毁时会自动调用析构函数

静态变量

  • 静态成员变量
    • 所有对象共享同一份数据
    • 在编译阶段分配内存
    • 类内声明，类外访问
  • 静态成员函数
    • 所有对象共享同一个函数
    • 静态成员函数只能访问静态成员变量
